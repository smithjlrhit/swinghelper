SwingHelper project

The goal of this project is to collect, test and discuss all kinds of debugging techniques for Swing

Every experienced Swing developer knows that Swing components must be accessed from Event Dispatch Thread (EDT) only,
working with JComponents from any other thread may lead to unpredictable result.

But how EDT rule violations can be found ?

org.swinghelper.debug.CheckThreadViolationRepaintManager will help you to find EDT rule violations in your code
