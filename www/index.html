SwingHelper project

The main goal of this project is to collect, test and discuss all kinds of debugging techniques for Swing

All code is tested with java 1.5 and 1.6, check the folders under src/java

org.jdesktop.swinghelper.debug.CheckThreadViolationRepaintManager
will help you to find Event Dispatch Thread (EDT) rule violations in your code

org.jdesktop.swinghelper.debug.EventDispatchThreadHangMonitor
will help you to find deadlocks and monitor the EDT for events that take longer than a certain time to be dispatched

in org.jdesktop.swinghelper.draft
you will find examples of some useful techniques for Swing debugging

org.jdesktop.swinghelper.transformer
is not about debugging, it is about Swing power and flexibility ;-)
